# Sistema de design

- **Tipografia:** Plus Jakarta Sans (400–800).
- **Cores:**
  - Navy `#2A1A4A` (fundos escuros e contornos)
  - Orange `#F58220` (ações e preço)
  - Ink `#1E1B3A` (texto principal)
  - Muted `#9A96B3` (texto secundário)
  - Line `#EAE6F2` (bordas)
- **Raios:** shell 32px em cards principais; inputs e botões com raio 12–16px.
- **Sombra:** `0 20px 60px rgba(42, 26, 74, 0.18), 0 6px 20px rgba(42, 26, 74, 0.12)` aplicada em cards principais.

Os valores vivem em `src/styles/globals.css` e estão mapeados no `tailwind.config.js` para uso direto nas classes utilitárias.
